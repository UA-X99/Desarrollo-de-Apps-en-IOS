# Proyectos

Esta carpeta contiene los proyectos principales desarrollados durante el curso.

## Estructura

Cada proyecto debe estar en su propia carpeta con un nombre descriptivo:
- `Proyecto01-NombreDelProyecto/`
- `Proyecto02-NombreDelProyecto/`
- etc.

## Contenido de cada proyecto

Cada carpeta de proyecto debe incluir:
- El proyecto Xcode completo
- Un archivo `README.md` con:
  - Descripción del proyecto
  - Características principales
  - Tecnologías utilizadas
  - Capturas de pantalla
  - Instrucciones de instalación y ejecución
  - Requisitos del sistema
