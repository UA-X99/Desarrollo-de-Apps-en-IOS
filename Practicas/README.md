# Prácticas

Esta carpeta contiene las prácticas realizadas durante el curso de Desarrollo de Apps en iOS.

## Estructura

Cada práctica debe estar en su propia carpeta con el siguiente formato:
- `Practica01-NombreDescriptivo/`
- `Practica02-NombreDescriptivo/`
- etc.

## Contenido de cada práctica

Cada carpeta de práctica debe incluir:
- El proyecto Xcode completo
- Un archivo `README.md` con:
  - Descripción de la práctica
  - Objetivos
  - Capturas de pantalla (si aplica)
  - Instrucciones de ejecución
