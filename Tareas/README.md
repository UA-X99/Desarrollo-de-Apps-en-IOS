# Tareas

Esta carpeta contiene las tareas y ejercicios asignados durante el curso.

## Estructura

Cada tarea debe estar en su propia carpeta con el siguiente formato:
- `Tarea01-Descripcion/`
- `Tarea02-Descripcion/`
- etc.

## Contenido de cada tarea

Cada carpeta de tarea debe incluir:
- El código fuente o proyecto Xcode
- Un archivo `README.md` con:
  - Enunciado de la tarea
  - Solución implementada
  - Notas adicionales
